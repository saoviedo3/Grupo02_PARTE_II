DOM Animación

Todas las animaciones deben ser relativas a un elemento contenedor. El elemento contenedor debe crearse con estilo = " position: relative". El elemento de animación debe crearse con estilo = " position: absolute". Las animaciones de JavaScript se realizan mediante la programación de cambios graduales en el estilo de un elemento. Los cambios son llamados por un temporizador. Cuando el intervalo del temporizador es pequeño, la animación parece continua. El código básico es:

function myMove() {
    let id = null;
    const elem = document.getElementById("animateanima");
    let pos = 0;
    clearInterval(id);
    id = setInterval(frame, 5);
    function frame() {
        if (pos == 200) {
          clearInterval(id);
          elem.style.transform = "rotate(0deg)";
          elem.style.backgroundColor = "red";
        } else {
          pos++;
          elem.style.top = pos + "px";
          elem.style.left = pos + "px";
          elem.style.transform = "rotate(" + pos + "deg)";
          elem.style.backgroundColor = "blue";
        }
    }
}

Eventos DOM

Un JavaScript se puede ejecutar cuando ocurre un evento, como cuando un usuario hace clic en un elemento HTML.

Para ejecutar código cuando un usuario hace clic en un elemento, agregue código JavaScript a un atributo de evento HTML:

onclick=JavaScript

Ejemplos de eventos HTML:

  • Cuando un usuario hace clic con el mouse
  • Cuando una página web ha cargado
  • Cuando se ha cargado una imagen
  • Cuando el mouse se mueve sobre un elemento
  • Cuando se cambia un campo de entrada
  • Cuando se envía un formulario HTML
  • Cuando un usuario pulsa una tecla

En este ejemplo, el contenido del boton cambia cuando un usuario hace clic en él:

function changeText(id) {
  id.innerHTML = "El texto ha cambiado";
}

DOM Oyente de Eventos

UEl addEventListener()método adjunta un controlador de eventos al elemento especificado y a un elemento sin sobrescribir los controladores de eventos existentes.

Para ejecutar código cuando un usuario hace clic en un elemento, agregue código JavaScript a un atributo de evento HTML:

  • Puede agregar muchos controladores de eventos a un elemento.
  • Puede agregar muchos controladores de eventos del mismo tipo a un elemento, es decir, dos eventos de "clic".
  • Puede agregar detectores de eventos a cualquier objeto DOM, no solo elementos HTML. es decir, el objeto ventana.

Al usar el addEventListener()método, el JavaScript se separa del marcado HTML, para una mejor legibilidad y le permite agregar detectores de eventos incluso cuando no controla el marcado HTML.

Puede eliminar fácilmente un detector de eventos utilizando el removeEventListener()método.

Sintaxis

element.addEventListener(event, function, useCapture);
  • El primer parámetro es el tipo de evento (como " click" o " mousedown" o cualquier otro evento HTML DOM ).
  • El segundo parámetro es la función que queremos llamar cuando ocurra el evento.
  • El tercer parámetro es un valor booleano que especifica si se debe usar el burbujeo de eventos o la captura de eventos. Este parámetro es opcional.

Nota

Tenga en cuenta que no utiliza el prefijo "on" para el evento; utilice " click" en lugar de " onclick".

En este ejemplo, el addEventListener()método le permite agregar muchos eventos al mismo elemento, sin sobrescribir los eventos existentes:

element.addEventListener("mouseover", Funcion1);
element.addEventListener("click", Funcion2);
element.addEventListener("mouseout", Funcion3);

Navegacion DOM

Con el DOM HTML, puede navegar por el árbol de nodos utilizando el nodo Relaciones.

Nodos DOM

De acuerdo con el estándar W3C HTML DOM, todo en un documento HTML es un nodo entonces:

  • Todo el documento es un nodo de documento
  • Cada elemento HTML es un nodo de elemento
  • El texto dentro de los elementos HTML son nodos de texto
  • Cada atributo HTML es un nodo de atributo (obsoleto)
  • Todos los comentarios son nodos de comentarios

Con el DOM HTML, JavaScript puede acceder a todos los nodos del árbol de nodos y se pueden crear nuevos nodos, y todos Los nodos se pueden modificar o eliminar.

En el siguiente ejemplo se recupera el texto de un elemento < h1> y lo copian en un elemento < p> 

document.getElementById("id02").innerHTML = document.getElementById("id01").innerHTML;

En el siguiente ejemplo se unen los conceptos

function addParagraph() {
var container = document.getElementById('container');
   var inputText = document.getElementById('inputText');
    var word = inputText.value;
   if (word.trim() !== '') {
        var newParagraph = document.createElement('p');
       newParagraph.innerHTML = word;
        var deleteButton = document.createElement('button');
       deleteButton.innerHTML = 'Eliminar';
       deleteButton.onclick = function() {
           container.removeChild(newParagraph);
           container.removeChild(deleteButton);
       };
        container.appendChild(newParagraph);
       container.appendChild(deleteButton);
        inputText.value = '';
   }
}

Colecciones DOM

El objeto HTMLCollection

El método devuelve un objeto HTMLCollection que es una lista similar a una matriz (colección) de elementos HTML

El código siguiente se puede seleccionar todos los elementos ya sean < p> u otros, de un documento

const myCollection = document.getElementsByTagName("p");

Y se pude acceder a los elementos de la colección mediante un número de índice 

myCollection[1]

Al presionar el botón se aplicará un estilo css a la colección de párrafos de todo el documento

Lista de Nodos DOM

Un NodeListobjeto es una lista (colección) de nodos extraídos de un documento.

Un NodeListobjeto es casi lo mismo que un HTMLCollectionobjeto.

Algunos navegadores (más antiguos) devuelven un objeto NodeList en lugar de HTMLCollection para métodos como getElementsByClassName().

Todos los navegadores devuelven un objeto NodeList para la propiedad childNodes.

La mayoría de los navegadores devuelven un objeto NodeList para el método querySelectorAll().

En este ejemplo, La lengthpropiedad define el número de nodos en una lista de nodos:

myNodelist.length
